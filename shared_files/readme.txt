NAMING SCHEME
id = RecNet image id
player = RR account id
room = RR room id (None for older pics)

Image name at the end is RR image server's name.

Images are ordered by date thanks to the naming scheme.

ACCESSING IMAGES FROM REC ROOM
You can access the original RecNet post via https://rec.net/image/{id}
You can access the original image from Rec Room's image server via https://img.rec.net/{image_name} (NOTE: 
you may have to omit the .png extension from older images)

FINDING PLAYERS / ROOMS
You can find players by id with RecNetBot: /profile data id:{player}
Same with rooms: /room data id:{room_id}

You can find a player's id with RecNetBot: /profile data username:{username} (if you wish to search tagged images by player)
Same with rooms: /room data name:{room_name}

RAW DATA
*_data folders contain the raw image data split by 1,000 images.
